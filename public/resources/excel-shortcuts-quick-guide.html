Excel Shortcuts Quick Guide

Simple shortcuts for beginners, students, and young professionals. Educational information only. Shortcuts are primarily for Microsoft Excel on Windows. Mac and web versions may differ.

Page 2 of 2 content included
Beginner-friendly

1. Movement and Navigation

Ctrl + Arrow
Jump to the edge of a data area.
Ctrl + Home
Go back to cell A1.
Ctrl + End
Go to the last used cell on the sheet.
Page Up / Page Down
Move one screen up or down.
Ctrl + PgUp / PgDn
Move between worksheet tabs.

2. Selecting Cells

Shift + Arrow
Extend your selection one cell at a time.
Ctrl + Shift + Arrow
Select to the edge of a data area.
Ctrl + A
Select the current table or worksheet.
Shift + Space
Select the entire row.
Ctrl + Space
Select the entire column.

3. Editing and Undo

F2
Edit the active cell without retyping it.
Esc
Cancel an entry or close a menu.
Ctrl + Z
Undo your last action.
Ctrl + Y
Redo or repeat your last action.
Ctrl + C / X / V
Copy, cut, and paste.
Ctrl + Alt + V
Open Paste Special.

4. Formatting Basics

Ctrl + 1
Open Format Cells.
Ctrl + B / I / U
Bold, italicize, or underline.
Ctrl + Shift + $
Apply currency format.
Ctrl + Shift + %
Apply percent format.
Ctrl + Shift + #
Apply date format.
Ctrl + Shift + ~
Apply general number format.

5. Formulas and Functions

Alt + =
Insert AutoSum.
F4
Lock references like $A$1 while editing a formula.
Ctrl + `
Show or hide formulas.
Ctrl + [
Jump to cells used by a formula.
Ctrl + ]
Jump to formulas using the selected cell.
Tab
Accept a suggested function or formula item.

6. Find, Save, and Print

Ctrl + F
Find text, values, or formulas.
Ctrl + H
Find and replace.
F5 or Ctrl + G
Open Go To for a cell or range.
Ctrl + S
Save the workbook.
Ctrl + P
Open print settings.
Ctrl + K
Insert or edit a hyperlink.

7. Rows, Columns, and Sheets

Ctrl + Plus
Insert cells, rows, or columns.
Ctrl + Minus
Delete cells, rows, or columns.
Ctrl + 9
Hide selected rows.
Ctrl + 0
Hide selected columns.
Ctrl + Shift + 9
Unhide selected rows.
Shift + F11
Insert a new worksheet.

8. Tables, Filters, and Data

Ctrl + T
Turn a range into an Excel table.
Ctrl + Shift + L
Turn filters on or off.
Alt + Down
Open a filter dropdown.
Alt, A, M
Open Remove Duplicates.
Alt, A, S, A
Sort ascending from the Data tab.
Alt, A, S, D
Sort descending from the Data tab.

9. Fast Data Entry

Ctrl + ;
Enter today's date.
Ctrl + Shift + ;
Enter the current time.
Alt + Enter
Start a new line inside a cell.
Ctrl + Enter
Fill selected cells with the same entry.
Ctrl + D
Fill down from the cell above.
Ctrl + R
Fill right from the cell to the left.

10. Workbook Management

Ctrl + N
Create a new workbook.
Ctrl + O
Open a workbook.
F12
Open Save As.
Ctrl + W
Close the active workbook window.
Ctrl + F1
Show or hide the ribbon.
Alt once
Show ribbon letters for keyboard commands.

11. Useful Alt Key Paths

Alt, H, O, I
Auto-fit column width.
Alt, H, V, S
Open Paste Special.
Alt, H, B
Open border options.
Alt, H, H
Open fill color options.
Alt, W, F, F
Freeze panes.
Alt, N, V
Create a PivotTable.

12. Common Quick Fixes

Arrow keys scroll sheet
Press Scroll Lock to turn it off.
Formula shows as text
Check for a leading apostrophe or text format.
Number signs (####) in a cell
Widen the column or change the number/date format.
Green error triangle
Click it to review Excel's warning.
Slow workbook
Remove extra formatting and unused rows and columns.
